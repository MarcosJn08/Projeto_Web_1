# Loja de Games

## Descrição
Este projeto consiste no desenvolvimento de um site de **Loja de Games**, com o objetivo de apresentar um catálogo de jogos, exibir trailers e permitir a simulação de compra através de formulários.

---

## 🚀 Tecnologias utilizadas

- **HTML5** — estrutura semântica das páginas
- **CSS3** — estilização customizada (`style.css`)
- **Bootstrap 5** — grid, componentes (dropdowns, carousel, cards, formulários) e utilitários
- **Bootstrap Icons** — ícones utilizados em toda a interface
- **JavaScript** (via `bootstrap.bundle.min.js`) — comportamento dos componentes interativos do Bootstrap (dropdowns, carousel)

## ✨ Funcionalidades

- **Página inicial**: caroosel com jogos em destaque e seção de cards com os mais vendidos/gratuitos.
- **Menu de navegação**: dropdown a partir do logo, com acesso a Início, Catálogo e Contato.
- **Barra de pesquisa**: campo de busca de jogos no cabeçalho, presente em todas as páginas.
- **Carrinho de compras**: cabeçalho com itens adicionados, valor total e botão para finalizar compra; página de cadastro/checkout com resumo de valores e seleção de método de pagamento (Pix ou cartão).
- **Notificações e conta**: notificações da conta do usuário.
- **Catálogo de jogos**: listagem de todos os jogos disponíveis.
- **Páginas de detalhes**: uma página individual para cada jogo com suas informações.
- **Página de contato**: formulário de envio de mensagem (nome, email, assunto, mensagem) e seção de perguntas frequentes (FAQ).
